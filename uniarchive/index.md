**UniArchive** — это сервис для централизованного хранения и поиска учебных материалов, решающий проблему потери файлов в разных мессенджерах и чатах.

Сервис построен на **микросервисной архитектуре** с асинхронным взаимодействием через **Apache Kafka** и предоставляет REST API для загрузки и интеллектуального поиска файлов.

## Ключевые возможности

*   **Единая точка загрузки**: API для передачи файлов с метаданными (предмет, название).
*   **Асинхронная обработка**: Автоматическое извлечение текста из загруженных файлов (PDF, DOCX) в фоновом режиме.
*   **Умный поиск**: API для фильтрации материалов по предмету и сортировки по дате.
*   **Масштабируемость**: Независимые микросервисы (File Upload, Text Extraction, Search API) общаются через события Kafka.

## Для кого этот сервис

*   **Студенты**: Чтобы быстро найти старые лекции и лабораторные работы.
*   **Преподаватели**: Для организации и распространения учебных материалов.
*   **Администраторы**: Для управления архивом и мониторинга системы.
