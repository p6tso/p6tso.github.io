# Документация: Визуальный дизайн и стилистика игры

## Стилистика
- **Стиль**: Child drawing (каракули)
- **Цветовая схема**: Чёрно-белая (BW)
- **Шрифт**: Bezier Sans
- **Тёмная тема**: Не используется

## Разработка визуального стиля

### 1. Создание стилистики
Использовалась комбинация:
- **Нейросетей** для генерации референсных изображений
- **Художника (Вики)** для финальной отрисовки

**Процесс:**
1. **Нейросеть** создавала изображения по промптам, например:
    - "A children's doodle style drawing of the main survivor base in a zombie apocalypse..."
    - "Simple, messy, and fun drawing style, like a child's crayon sketch."
    - С уточнениями: "very simple, naive, and charmingly crudely drawn with squiggly lines"

2. **Результат**: множество детских рисунков для референсов.

3. **Художник (Вика)** использовала референсы для понимания нужного стиля и отрисовки итоговых изображений.

### 2. Создание анимации
- **Изначальный план**: Использование GIF-анимации.
- **Финальное решение**: Встроенный движок (Phaser) для покадровой анимации.
- **Процесс**: Аниматор (Григорий) рендерил анимации в After Effects как раскадровку (несколько кадров).

### 3. Дизайн UI
- **Основа**: Минимализм.
- **Цель**: Сделать акцент на геймплее.
- **Референсы**: Современные сайты с мини-играми (например, abstract).
- **Цветовая основа**: Белый цвет.
- **Преимущество**: Белый фон позволяет выделять важные элементы (например, игровой баланс) любым цветом, привлекая внимание пользователя.
- **Подход**: Ориентация на выигрышные решения конкурентов/других сайтов.
- **Результат**: Приятный, практичный и минималистичный дизайн.
