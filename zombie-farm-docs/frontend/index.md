# Frontend-архитектура

## Технологический стек

### Основные инструменты
- **Vite** — современный сборщик проекта
    - Обеспечивает быструю сборку и горячую перезагрузку (HMR)
    - Обеспечивает реактивное обновление страниц при изменениях кода
    - Используется как инструмент разработки и сборки

- **React** — основной UI-фреймворк
    - Используется для построения пользовательского интерфейса
    - Управляет состоянием приложения (кроме игровой логики)
    - Обрабатывает меню, интерфейсы, панели управления

- **Phaser 3** — игровой движок
    - Обрабатывает игровую логику и графику
    - Работает с игровыми объектами, анимациями, физикой
    - Поддерживает JavaScript и TypeScript
    - Имеет активное сообщество и документацию

### Особенности Vite
- Мгновенная горячая перезагрузка (HMR)
- Оптимизированная сборка для production
- Поддержка TypeScript из коробки
- Плагины для React и других библиотек
